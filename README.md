## Sistema de Cadastro em C#

Aplicação CRUD desenvolvida em C# com integração ao banco de dados MySQL

## Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Remoção de usuários

## Tecnologias
- C#
- .NET
- MySQL

## Estrutura
- Models
- Data
- Services

## Objetivo
Projeto desenvolvido para prática de desenvolvimento backend e integração com banco de dados.

## Como executar

1. Execute o script `database.sql` no MySQL
2. Configure a string de conexão no arquivo Conexao.cs
3. Execute o projeto no Visual Studio
